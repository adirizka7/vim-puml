*vim-puml*    Plugin for generating PlantUML diagrams

NAME
    vim-puml - Generate and view PlantUML diagrams within Vim.

COMMANDS
    :PUMLGenerate
        Generates a PlantUML diagram from the current `.puml` file.

    :PUMLOpen
        Opens the generated diagram in the default browser.

    :PUMLShare
        Copies the PlantUML share url for the current `.puml` file into the `+`
        register.
